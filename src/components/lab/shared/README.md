# Shared Lab Components
هذا المجلد مخصص لمكونات واجهة المستخدم المشتركة (Shared UI) بين MT5 و OKX.
- مسموح فقط بمشاركة المكونات التصميمية البحتة (مثل الأزرار، التنسيقات، الألوان).
- ممنوع قطعاً مشاركة أي منطق تداول (Trading Logic) أو مؤشرات (Indicators) أو إدارة مخاطر (Risk Management) في هذا المجلد.
